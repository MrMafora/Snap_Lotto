How The Scanner Works

Ticket Scanning Process:

1
User uploads lottery ticket image or takes photo using their device camera
2
Google Gemini API processes the image using OCR to extract ticket information
3
System checks extracted numbers against our database of lottery results
4
If draw information is missing, OpenAI API fetches the official results
5
Results are presented to the user showing matches and potential winnings
