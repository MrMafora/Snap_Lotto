Lottery Results
GET /api/v1/results

Get latest lottery results

Parameters: lottery_type, limit, date_from, date_to
curl -H "Authorization: Bearer YOUR_TOKEN" \ "http://localhost:5000/api/v1/results?lottery_type=LOTTO&limit=5"
GET /api/v1/results/{lottery_type}

Get results for specific lottery type

curl -H "Authorization: Bearer YOUR_TOKEN" \ "http://localhost:5000/api/v1/results/PowerBall"
Analytics
GET /api/v1/analytics/frequency

Get number frequency analysis

Parameters: lottery_type, days
curl -H "Authorization: Bearer YOUR_TOKEN" \ "http://localhost:5000/api/v1/analytics/frequency?lottery_type=LOTTO&days=365"
GET /api/v1/analytics/predictions

Get AI-powered predictions

Parameters: lottery_type (required)
curl -H "Authorization: Bearer YOUR_TOKEN" \ "http://localhost:5000/api/v1/analytics/predictions?lottery_type=LOTTO"
